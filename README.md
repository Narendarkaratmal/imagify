# 🎨 Imagify – AI Image Generator

An AI-powered web application that generates images from text prompts using advanced APIs.  
It leverages the ClipDrop API for image generation and Imagify for image optimization.

---

## 🚀 About the Project

Imagify is a full-stack web application that allows users to create high-quality images from text descriptions.  
The platform integrates AI services to deliver fast, optimized, and visually appealing results.

It also includes authentication and a credit-based system to manage user access efficiently.

---

## 🧠 Features

- 🧾 Generate images from text prompts (AI-powered)  
- 🔐 User authentication with JWT  
- 💳 Credit-based usage system  
- ⚡ Fast image processing and optimization  
- 🎨 Clean and responsive UI

## 🔌 APIs Used

- ClipDrop API – Image generation  
- Imagify API – Image optimization  

---



## ⚙️ How It Works

- User enters a text prompt  
- Request is sent to ClipDrop API  
- Generated image is optimized using Imagify  
- Credits are deducted per image generation  
- Final image is displayed to the user
